Die Berechtigung für den Ordner sollte so aussehen:
![berechtigung_ssh_ordner.png](/berechtigung_ssh_ordner.png)
Mit folgendem Befehl kann diese eingerichtet werden:
```bash
chmod 700 .ssh
```
